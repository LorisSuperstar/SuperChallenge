# SuperChallenge 
- Inspiration from Bastighg Challenges.
- Modloader: Fabric
- Minecraft-Version: 1.21
- Mappings: Yarn

## Download instructions
1. Download the File
2. Put the File into the Mods Folder
3. Put the right Fabric Version in.

## Challenges
- Damage = Fly
- OneChunk 
- Only Dirt

## Ingame Instructions
You need Creative or Commands
### Creative
Open the CreativeInventory with E and go to the Operator Utilities Tab. There you will find Items that you can Right Click to Activate.
### Commands
/give @s challenges:damagefly

/give @s challenges:onechunk

/give @s challenges:onlydirt
